🛍️ Shopping Mall Visitor Analysis

📋 Overview
This project aims to analyze visitor behavior in shopping malls based on variables such as gender, age, annual income, and spending score.
Through exploratory data analysis and data visualization techniques, the project reveals insights that help better understand customer profiles and preferences.

🚀 Main Features

✔ Exploratory Analysis: Detailed exploration of variables and their relationships.
✔ Interactive Visualizations: Charts highlighting behavioral patterns.
✔ Business Insights: Identification of visitor segments based on income and spending score.

🧰 Technologies Used

Python

Pandas — Data manipulation and cleaning

Matplotlib / Seaborn — Data visualization

NumPy — Numerical operations support

Scikit-Learn — Used for clustering tests (in future versions)

⚙️ How to Run

1️⃣ Clone the repository:

git clone https://github.com/rodrigohigashi/Data_Analysis.git
cd Data_Analysis


2️⃣ Create a virtual environment and install dependencies:

python -m venv venv
venv\Scripts\activate  # Windows
# or
source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt


3️⃣ Run the notebook:

jupyter notebook notebooks/EN/Visitor_Analysis.ipynb

📊 Key Insights

📌 Female visitors represent the majority of the audience.
📌 The predominant income range is between $50,000 and $85,000 annually.
📌 The analysis shows a positive relationship between annual income and spending score.

🎯 Takeaways

This project reinforces the importance of exploratory data analysis as an essential step to understand consumer behavior and support more effective marketing strategies.
